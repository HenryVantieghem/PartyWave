### Screenshots

**iPhone 6.7" Display (1290 x 2796 pixels)** - Required:
1. **Hero Screen**: Party Radar with proximity circles (main value prop)
2. **Crew System**: Crew detail with member list
3. **Party Detail**: Live party pulse with energy meter
4. **Party Creation**: Beautiful create party flow
5. **Passport**: Stats grid with gamification
6. **Memories**: Photo grid with highlights

**iPhone 6.5" Display (1242 x 2688 pixels)** - Required:
- Same content as 6.7" but resized

**iPhone 5.5" Display (1242 x 2208 pixels)** - Optional:
- Same content but resized for smaller display

**iPad Pro 12.9" Display (2048 x 2732 pixels)** - Optional:
- Optimized layouts for tablet view

**Screenshot Design Tips**:
- Use real-looking mock data (not lorem ipsum)
- Show the app in action (people using features)
- Add subtle background gradients
- Include short, punchy text overlays
- Maintain brand colors (black, coral, purple)
- Show diversity in user avatars
- Highlight unique features (proximity circles, live pulse)

### App Preview Videos

**Specifications**:
- Length: 15-30 seconds
- Format: M4V, MP4, or MOV
- Resolution: 1080p or 4K
- Frame rate: 25-30 fps
- Aspect ratio: 16:9 or 9:16 (portrait)

**Video Storyboard**:
1. **0-5s**: Open on Party Radar - show proximity circles expanding
2. **5-10s**: Tap party, show immersive detail with parallax header
3. **10-15s**: Show live party pulse updating in real-time
4. **15-20s**: Quick transition to crew screen, show activity feed
5. **20-25s**: End on tagline "Your Vibe. Your Crew. Your Night."
6. **25-30s**: Logo reveal with app name
